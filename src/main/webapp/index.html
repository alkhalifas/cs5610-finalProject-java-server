Welcome to the CodeSnippets API

Please select from the following endpoints:

  • /api/snippets

    Which displays a list of all snippets

  • /api/snippets/123

    Which displays snippet 123

  • /api/creators/creatorId/snippets

    Which displays a list of all snippets for a specific creator

  • /api/snippets/public

    Which displays a list of all snippets that are public

  • /api/users

    Which retrieves all users

  • /api/users/123

    Which displays user 123
